                            function getValues() {
                                let inputSting = document.getElementById('userString').value

                                let revString = reverseString(inputSting);
                                displayString(revString);
                            }

                            //logic function
                            // reverse a string
                            function reverseString(userString) {
                                //reverse the string
                                let newSring = ''

                                for (let i = userString.length -1; i >= 0; i--) {
                                    newSring += userString[i];
                                }

                                return newSring;
                            }

                            //view function
                            function displayString(revString) {
                                document.getElementById('results').textContent = revString;
                                document.getElementById('alert').classList.remove('invisible');
                            }

The code is structured in three functions

FunctiongetValues: grabs the user input. Then reverseString function is called and the value received from user is passed to this function. Once value from reverseString function is received displayString function is called to display the result.

Function reverseString reverses the string that was received as a parameter. I declare an empty string variable to assign a reverse string. I am using a for loop to iterate through each element of the string, each element will be assigned to a new string variable. The number of characters in the string is the number of times this loop will run. The start point is the last element of the string. Ones code is out of the for loop function retunes the reversed string.

Function displayStringreceives the value of the reversed string. Then the function grabs the element where reversed string will be displayed, and assigns it to that element. Then 'invisible' class is removed from alert, so that reversed string is visible as an alert.
